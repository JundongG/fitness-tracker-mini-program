# 健身跟踪管理小程序 (Fitness Tracker Mini-Program)

## 项目概述
一个专注于健身爱好者的移动端小程序，用于记录、跟踪和管理日常健身活动、进度和目标。

**目标用户**：健身新手到中级爱好者，希望简单高效跟踪训练的个人用户。

**核心价值**：
- 轻松记录每次训练
- 可视化进步
- 保持动力（目标、历史、提醒）
- 简单易用，无需复杂设备
